Operation Exodus

A collaborative team project between myself and fellow students for the DES310 module. Set on an overrun spaceship, Operation Exodus is a 2D top-down shooter created in the Unity Engine. My main contributions for this project were programming the mechanics for the player controller, weapon systems, the shop and upgrade systems, lootable crates, and item pickups.
