Station Sentinels

May-2023
C++  Box2D  Game  Procedural Generation 

Station Sentinels is a top down shooter set in randomly generated levels upon a space station crawling with evil robots.

Shoot guns and throw weapons to progress through the three floors of the main mode, or try your hand at setting a local high score in the infinite arcade mode.

My main contribution to this project was the weapon system, including the programming of all weapon functionality, being able to throw weapons, handling of bullet logic and explosion logic for rockets.

I also created the arcade mode for the game, along with difficulty settings, allowing for greater replayability and ways for players to challenge themselves further.

In addition, I was responsible for optimising the memory usage of the game by improving model loading along with fixing issues with the procedural level generation to make levels generate much faster and ensure that all generated rooms are connected to each other.

Station Sentinels was created using Abertay University’s Games Education Framework, programmed in C++ and using Box2D to handle physics and collisions.
